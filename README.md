Este projeto é uma calculadora gráfica desenvolvida em Java, utilizando a biblioteca Swing para a interface gráfica. 
O layout foi inspirado no estilo das calculadoras da Apple, com foco em simplicidade, usabilidade e visual moderno.

Operações básicas:
➕ Adição.
➖ Subtração.
✖️ Multiplicação.
➗ Divisão.
📊 Cálculo de porcentagem.
🧹 Botão AC para limpar tudo.
⌫ Botão de apagar último caractere.

Funcionamento do Código:
A classe Calculadora herda de JFrame e implementa ActionListener.
O display é um JTextField configurado como não editável.
Os botões são organizados em um GridLayout (5x4).
As operações são controladas por um switch com base no operador selecionado.
Os valores são armazenados em double para permitir cálculos decimais.

Desenvolvido para fins de estudo e prática com Java Swing.
